## 项目介绍

一个前后端分离的甜品商城系统，用户可以浏览甜品、加入购物车、下单购买；管理员可以在后台管理商品、用户和订单，并查看数据统计。系统界面美观，功能完整，适合作为毕业设计或课程设计参考。

## 功能模块

- **用户登录注册**：支持用户注册、登录，区分普通用户和管理员角色
- **商品浏览**：首页展示热门甜品，支持按分类筛选和关键词搜索
- **购物车**：商品加入购物车，支持数量修改和删除
- **订单管理**：用户下单、查看订单列表和订单详情，支持多种订单状态
- **个人中心**：查看和编辑个人资料
- **管理后台仪表盘**：用户数、订单数、销售分类统计、订单状态分布等数据可视化
- **甜品管理**：管理员对商品进行增删改查，支持分类筛选和搜索
- **用户管理**：管理员查看和管理系统用户
- **订单管理**：管理员查看所有订单，处理订单状态

## 技术架构

- **后端**：Spring Boot 2.6.13 + MyBatis-Plus 3.5.3 + Lombok
- **前端**：Vue 3.5 + TailwindCSS 4 + Vite 7 + ECharts 6 + Pinia 3
- **数据库**：MySQL 8.0
